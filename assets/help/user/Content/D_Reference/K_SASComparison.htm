SAS-equivalent functions

This section lists the Xcalar functions that accomplish the same tasks as some of the most commonly used SAS functions. If an equivalent Xcalar function is not available, this section suggests an alternate method. Also, if the Xcalar function works differently than the corresponding SAS function, a brief description of the Xcalar function is provided.

SAS function category: Arithmetic

SAS function SAS description Xcalar Map function Xcalar syntax
ABS Returns the absolute value of a number. abs map(abs(operand))
MOD Returns the remainder from division. mod map(mod(operand1, operand2))
SQRT Returns a positive square root. sqrt map(sqrt(operand))

SAS function category: Character

SAS function SAS description Xcalar Map function Xcalar syntax
COMPBL Removes multiple blanks between words in a character string. replace map(replace(string, " ", ""))
COMPRESS Removes specific characters from a character string. replace map(replace(string, char, ""))
INDEX Searches the source for the character string specified by the excerpt. find

map(find(string to search, string to find, starting index, ending index))

The search is case-sensitive.
LENGTH Returns the number of characters in a text string. len map(len(operand))
SUBSTR Returns the specified number of characters starting at the specified position. substring map(substring(string, start_index, end_index))
TRIM Removes trailing blanks from character expression and returns one blank if the expression is missing. strip

map(strip(operand))

Strip leading and trailing white space from a string.

SAS function category: Mathematical

SAS function SAS description Xcalar Map function Xcalar syntax
EXP Returns the value of the exponential function. It raises the constant e, which is approximately 2.71828, to the power that is supplied by the argument. power

map(power(operand1, operand2))

Use e as operand1.
LOG Returns the natural logarithm (base e). log

map(log(operand))
LOG2 Returns the base-2 logarithm of a number. log2

map(log2(operand))
LOG10 Returns the base-10 logarithm of a number. log10 map(log10(operand))
