Easy steps for importing data (with default settings)

This section describes the steps up to the point where you can start performing operations in tables presented in a workbook.

IMPORTANT: The easy steps are appropriate if you want to get started quickly to familiarize yourself with Xcalar. When you are ready to use Xcalar Design in a production environment, or if any of the assumptions listed in this section is not true, follow the instructions in Selecting a data source.

This section assumes that:

  • the data format of your data source is natively supported by Xcalar.

    NOTE: Unicode is supported for both delimiter-separated values and JSON that is UTF-8 encoded. UTF-16 or other encoding systems are not supported without UDFs.
    NOTE: Importing Excel data does not require you to specify a UDF, but Xcalar actually uses a UDF when importing the data.

  • field names in the data source do not contain illegal characters, which are the following characters.

    • caret (^)
    • single quote mark (')
    • double quote mark (")
    • left or right parenthesis (( ))
    • left or right square bracket ([ ])
    • left or right brace ({ })
    • backslash (\)
    • colons (:)
    • comma (,)
    • period (.)
  • you accept all the default settings suggested by Xcalar Design.

Importing data from the data source

Follow these steps to import data:

  1. Click (Datasets icon). Its location is shown in the following screenshot:

  2. In the Import Data Source window, follow these steps:

    1. Select the data target for accessing the data source. For information about data targets, see Understanding data targets.

      The following screen illustrates the Import Data Source window.

    2. In the Data Source Path field, type the path to the data source file. The path is relative to the data target.

      For example, if the mount point for the data target is /flightData/2017, and the complete path to the data source is /flightData/2017/airlines.csv, enter airlines.csv in the Data Source Path field.

      Xcalar Design accepts a path with or without the preceding forward slash (/).

    3. Click BROWSE to display the Browse Data Source window, in which you can navigate to the data source. Click to select the file or directory to be used as your data source, then click NEXT.

      NOTE: If your data source is a directory, Xcalar recommends that you follow the steps in Selecting a data source, which describes various options specifically for importing a directory.

  3. When a preview of the records in the data source is displayed, create a dataset by clicking CREATE DATASET. The Dataset Preview window is displayed.

You now have a dataset that imports data from a data source. The dataset is listed in the Datasets section of the left panel.
