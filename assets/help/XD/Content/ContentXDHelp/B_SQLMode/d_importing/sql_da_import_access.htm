Importing data into Xcalar Design

The following topics provide an abridged set of instructions for importing your data into Xcalar Design. For the complete set importing instructions, see the Getting Started section.

Preparing the data source

Before importing your data into Xcalar Design there are some data preparation considerations and guidelines that Xcalar recommends for an error free import. For more information, see Preparing the data source.

Selecting a data source

NOTE: The following procedure assumes that you are logged into Xcalar Design and that you are in a Workbook.

To select a data source:

  1. Verify that the post-installation tasks are completed and that you have access to your source data through a data target.
  2. Verify that you are in SQL Mode. If not, from the top menu bar of Xcalar Design, click the mode switch to SQL MODE .
  3. From the sidebar, click the Sources icon, which expands to reveal the Tables icon.
  4. Click the Tables icon and then in the Tables panel, click the Import Data Source icon.
    5. TIP: If you accidentally click the View IMD icon, you can switch back to the Tables panel by clicking on the View Tables icon in menu bar of the Published Tables section.
  5. From the Import Data Source panel in the Data Source Path, select Browse.

    6. The Browse Data Source window opens.

  6. From the file browser, locate your source data. For more information on browsing for your data, see Using the file browser to select the data source.
    7. IMPORTANT: You can import multiple objects, such as a folder or a selection of files, that contain the data for analysis into Xcalar. Before selecting multiple objects, verify that each object adheres to the same format and schema.
    NOTE: You can only import data from one worksheet of an Excel workbook.
  7. Do one of the following:
    • For a single file, double-click the file, which takes you directly to the Import Data Source window.
    • For multiple file objects:
      1. For each file object that contains your data, select its check box.
        2. NOTE: You can select multiple objects that contain your data. As you select or deselect files and/or file objects, the name and path of the file and/or file object is added to the Selected Files / Folders section. For more information, see Selecting multiple items as a data source.
      2. When you are satisfied with the list of files or file objects under the Selected Files / Folders section, click Next.

        3. The Import Data Source window opens.

For more information on the features and functions in the Browse Data Source window, see Selecting a data source.
