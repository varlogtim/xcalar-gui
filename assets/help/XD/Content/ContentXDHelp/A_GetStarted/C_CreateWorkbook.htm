Creating and activating a workbook

This topic describes how to create and activate a workbook in Xcalar Design. An active workbook is required for you to import data to Xcalar.

Before creating a workbook

Before creating a workbook, decide on the workbook name. Workbook names can be changed at any time.

The following list describes the naming conventions:
  • The name is case-sensitive.
  • Length must not exceed 255 characters.
  • Characters must be A-Z, a-z, 0-9, hyphen (-), underscore (_), or space.
  • First character must be a letter.
  • The name must not be one of the following strings:
    • udf
    • udfs
  • For the admin user, the name must not be PreUpgrade.

Creating a workbook

To create a workbook:

  1. The Workbook Browser is automatically displayed after you log in to Xcalar Design.
    If it is not displayed, click in the upper left corner of the window. In the New Workbook card, click Create New Workbook. A new card is displayed.

  2. To name the new workbook and add a description, do the following:

    1. Click the ellipsis icon and then click Edit Name and Description.

      The following list describes the naming conventions:
      • The name is case-sensitive.
      • Length must not exceed 255 characters.
      • Characters must be A-Z, a-z, 0-9, hyphen (-), underscore (_), or space.
      • First character must be a letter.
      • The name must not be one of the following strings:
        • udf
        • udfs
      • For the admin user, the name must not be PreUpgrade.
    2. In the Edit Workbook modal window, enter the workbook name.

    3. Enter a workbook description. This description is displayed on the workbook card in the Workbook Browser, enabling you to make the correct workbook selection.

    4. Click Confirm.
    TIP: If the complete description is too long to fit on the workbook card, move the cursor over the description to display a tooltip with the complete description.

Activating a workbook

Once a workbook is created, it is inactive. You must activate it before you can use it. For more information about workbook activation, see Managing workbooks.

Follow these steps to activate a workbook:

  1. Click anywhere in the card for the inactive workbook.
  2. When the ACTIVATE WORKBOOK modal window is displayed, click Confirm.
