Working with a workbook

Creating a workbook is the first step in designing your data model in Xcalar Design. Workbook can be visualized as a container where you design your data models and execute them. You can perform a number of tasks on a workbook.

For the following workbook tasks, see Creating and activating a workbook:

  • Creating a workbook
  • Adding a workbook name and description
  • Activating a workbook

For the following workbook tasks, see Managing workbooks:

  • Switching between workbooks
  • Duplicating a workbook
  • Downloading a workbook
  • Uploading a workbook
  • Deactivating a workbook
  • Deleting an inactive workbook
