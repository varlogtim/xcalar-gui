Importing data into Xcalar Design

The following topics provide an abridged set of instructions for importing your data into Xcalar Design with links to detailed information at every step. It is recommended that you read the following sections before starting to import your data:

  • Xcalar Design overview
  • Accessing Xcalar Design
  • Understanding the data source format requirements

Selecting between a dataset and a table

Before importing your data into Xcalar Design, you must select whether you want to import your data into a dataset or into a table. If you are going to work in the SQL Mode, you must import your raw data into tables. If you do not know the format of the raw data, you must import it into a dataset and then extract the parts you want in your dataflow.

Dataflow Mode enables you to import data in tables as well as datasets and SQL Mode enables you to import data in tables only. Depending on whether you want to import data in tables or datasets, verify that you are in the right mode. If not, from the top menu bar of Xcalar Design, click the mode switch to go to the right mode.

Selecting a data source

NOTE: The following procedure assumes that you are logged into Xcalar Design and that you are in a workbook.

To select a data source:

  1. From the sidebar, click the Sources icon , which expands to reveal the Datasets and the Tables icons.
  2. If you want to import the data into a table, click the Tables icon otherwise click the Datasets icon and then click the Import Data Source icon.

    3. The Source page opens.

  3. Select the connector, containing your data. For detailed information on selecting a connector, see Selecting a connector.

  4. In the Data Source Path section do one of the following:

    • Enter the path of your data and click Next.

    • Click Browse and from Browse Data Source window, select your source data.

      • For a single file, double-click the file, which takes you directly to the Format window.
      • For multiple file objects:
        1. For each file object that contains your data, select its check box.

        2. When you are satisfied with the list of files or file objects in the Selected Files / Folders section, click Next.

          IMPORTANT: You can import multiple objects, such as a folder or a selection of files, that contain the data for analysis into Xcalar. Before selecting multiple objects, verify that each object adheres to the same format and schema.
    NOTE: If you have chosen Database or Generated connector, you do not need to enter a path. For detailed information about selecting a data source, see Selecting a data source.

    5. The Format window opens. The format window displays the preview of the selected file and enables you to select the format and various other specifications.

  5. In the Format window, change the format specifications as per your requirements and click Create Table or Create Dataset. For detailed information about formats and their specifications, see Selecting format options during an import.

  6. On successful import, Xcalar creates the table or the dataset and displays its preview.

NOTE: If your data source is a parquet file or parquet dataset, follow the additional instructions in Importing from a Parquet file or dataset.
NOTE: Xcalar does not make a copy of source data files on the Xcalar cluster.
