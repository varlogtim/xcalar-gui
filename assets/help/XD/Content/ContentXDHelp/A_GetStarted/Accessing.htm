Accessing Xcalar Design

Before accessing Xcalar Design, make sure that the following requirements are met:

  • You use one of the following web browsers:

    • Google Chrome version 65 or later
    • Microsoft Edge version 44 or later

    Also, Xcalar Design does not run on mobile devices such as tablets or cell phones.

  • The source files containing the raw data to be analyzed are accessible to Xcalar Design.

The minimum recommended screen resolution for Xcalar Design is 1024 x 768.

Starting Xcalar Design

To start Xcalar Design, point the browser using the HTTPS protocol to the location where Xcalar Design is installed, as in the following example:

https://xcalar.mycompany.com/index.html

Understanding the Xcalar Design login

Log in to Xcalar Design with the login name and password obtained from the Xcalar administrator or by using the Microsoft Azure account credentials.

NOTE: Login names are case-insensitive.

You can have multiple logins at any given time. For example, after logging in from one browser tab, you can log in again from another browser tab. However, you cannot open the same workbook simultaneously from multiple browser tabs.

Suppose you are logged in to Xcalar Design from two browser tabs. After you open a workbook in one browser tab, your attempt to open the same workbook in another browser tab generates a warning message. You can either go back to the previous screen or continue opening the workbook in the second browser tab. If you continue, the workbook in the first browser tab is closed.

Understanding the license key

For all Xcalar Design features to be available, the Xcalar administrator must enter a valid license key during the installation. Without a valid license, you cannot perform SQL queries, execute dataflows, and so on. You can, however, perform a limited number of tasks, such as downloading your workbooks and exporting data from Xcalar Design. To ensure uninterrupted services, make sure that you have a valid license at all times.

If you cannot use Xcalar Design due to an invalid license, contact your Xcalar administrator as soon as possible. The Xcalar Design administrator can update the license key as described in Updating the license.

Displaying the product version and the license information

To display the Xcalar product version and the license information:

  1. In the upper right corner of the Xcalar Design window, click the user icon .

  2. In the drop-down menu, click About to display a modal window that lists the product version, the license key, and the license expiration date. If the Expiration field displays the text Unlicensed, the cluster does not have a valid license key.
