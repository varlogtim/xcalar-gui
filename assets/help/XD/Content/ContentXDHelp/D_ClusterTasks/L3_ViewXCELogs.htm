Displaying Xcalar Compute Engine logs (admin only)

Xcalar Compute Engine generates log messages and saves them to log files. By default, the log files are saved in the directory named /var/log/xcalar.

You can use Xcalar Design to view log messages generated by each node, which are in a file named /var/log/xcalar/node.X.log, where X is the node number. For example, on a two-node cluster, the log directory contains /var/log/xcalar/node.0.log and /var/log/xcalar/node.1.log.

NOTE: If you want Xcalar logs to be saved at a different location, contact Xcalar for instructions on changing the path of the log directory.

Methods for viewing logs

You can view log messages through Xcalar Design in one of the following two ways:

  • View a specified number of recent messages
  • Stream messages as they are generated

Specifying the nodes from which you retrieve the logs

You can view messages from all nodes or from specific nodes. To retrieve the logs from specific nodes:

  1. Display the Administration panel under System. The Logs window is displayed below the Configure Parameters window.
  2. In the Hostname Pattern field, enter a regular expression to match the hostnames. If a node has a hostname that matches the regular expression, its log will be displayed. If you leave this field blank, logs from all nodes will be displayed.

Viewing recent log messages

To view recent log messages:

  1. Display the Administration panel under System. The Logs window is displayed below the Configure Parameters window.

  2. In the Logs window, enter a log file name in the Log Name field. To display messages for all nodes, you can type the following:

    node.*.log

  3. Enter the number of messages from each log file to display, ranging from 1 to 500. For example, if you enter 10, then 10 messages from each log file are displayed.
  4. Click the check mark icon to submit the values. The specified number of messages for each log file appear in the window immediately.

Streaming log messages

To view log messages as they are generated:

  1. Display the Administration panel under System. The Logs window is displayed below the Configure Parameters window.

  2. In the Logs window, type the Log Name. To display messages for all nodes, enter the following:

    node.*.log

  3. Click Start Log Stream. The log messages from the specified log files are displayed in real time.
  4. If you want to stop the streaming, click Stop Log Stream.
NOTE: If you previously set a number of messages to display, that value has no effect on message streaming.

Clearing log messages

To remove all log messages from the Logs window, click Clear Logs. This button clears the messages from Xcalar Design only. The messages remain in the log files on the nodes.
