Using UDFs

This section describes the tasks related to UDFs (user-defined functions). To see examples of UDFs, see User-defined function (UDF) examples.

Creating a UDF in Xcalar Design

UDFs created by one Xcalar Design user can be read and copied by all other Xcalar Design users, so they can accomplish similar tasks.

You define one or multiple UDFs in a Python module. You can create a module in these ways:

  • Enter the code directly in the User Defined Function panel.
  • Upload Python code from your computer to the File Manager.
  • Use the JUPYTER NOTEBOOK window to display a UDF template and then use the Jupyter notebook editor to modify the code in the template. For more information about using the Jupyter Notebook, see Using Xcalar Design with Jupyter Notebook.

Entering the code directly

Follow these steps to write the UDF code directly in User Defined Function panel and then save it:

  1. Click the UDF icon () towards the bottom of the sidebar to display the User Defined Function panel. 
  2. Click the drop-down arrow to display a list of existing modules. Select one to use it as a template. Alternatively, select New Module if you want to start with an empty module.
  3. Enter the code in the panel.

  4. Click on the Save File icon () on the top to save the code. A modal SAVE AS window appears, where you can type in a new file name with extension py.

    NOTE: Module names are case insensitive. If you type letters in uppercase, Xcalar Design displays them in lowercase when listing the modules. See UDF naming conventions for information about name requirements.

  5. If a module with the same name already exists, a dialog box asks whether you want to overwrite the existing module. If so, click Confirm. Otherwise, click Cancel and then rename the module.

Note: Overwriting a functioning UDF with a non-functional one can cause the execution of some operations (for example Map or import which invoke the UDF) to fail. The best practice is to apply disciplined versioning to your UDFs.
TIP: When editing a UDF, you can use the keyboard shortcut appropriate for your operating system to search for or replace a string. For example, if you run Xcalar Design in a browser on Microsoft Windows, press the Ctrl and F keys to search for a string, and the Ctrl, Shift, and F keys to replace a string.

Uploading a UDF module

Follow these steps to upload a file as a UDF module from your computer to your cluster:

  1. Click the UDF icon () towards the bottom of the sidebar to display the User Defined Function panel.
  2. Open the File Manager by clicking on the Open File Manager icon () in User Defined Function panel. File Manager window opens up and displays the existing UDF modules.

  3. Click on the Upload button in the top right corner.

    NOTE: The Upload button may be disabled if you navigate to a UDF folder where you do not have write permission.

  4. Navigate to the folder on your computer which contains the file you want to upload. Select a file with extension py or txt. If the file has valid Python code, it is uploaded and the file name with extension py appears in the File Manager, otherwise an error is reported.

  5. If a file with the same name already exists, it is replaced with the new file.

    NOTE: If the original file name does not follow the naming conventions, an error message notifies you that the name is invalid. See UDF naming conventions for naming requirements.

Downloading a UDF module

Follow these steps to download a UDF module from your cluster to your computer:

  1. Click the UDF icon () towards the bottom of the sidebar to display the User Defined Function panel.
  2. Click the Open File Manager icon () in the User Defined Function panel.
  3. Navigate to the desired folder by double clicking on the folder name and select the desired UDF file.
  4. Click Download from the Action drop-down menu to download the desired module to your computer.

Editing a UDF module

Follow these steps to edit an existing UDF module:

  1. Click the UDF icon () towards the bottom of the sidebar to display the User Defined Function panel.
  2. Select the Open File Manager icon () in User Defined Function panel. A list of modules is displayed.
  3. Select the desired module.
  4. Edit the module in one of the following ways:
    • Click Open from the Action drop-down menu to edit the code in User Defined Function panel.
    • Click Download from the Action drop-down menu to download the desired module to your computer. Modify the module by using an editor on your computer. Upload the file after you finish editing.
    • Click Open from the Action drop-down menu to display the code in User Defined Function panel. Copy the desired module code to Jupyter Notebook or external UDF editor. Paste new module code back into the UDF editor.
  5. Click Save icon () to save it. Optionally, add a version number string to the module name and then save it.

Deleting a UDF module

IMPORTANT: Be sure you understand the effects of deleting a UDF module before you proceed. You cannot undo UDF module deletion.

Follow these steps to delete a UDF module:

  1. Click the UDF icon () towards the bottom of the sidebar to display the User Defined Function panel.
  2. Select the Open File Manager icon () in the User Defined Function panel. A list of modules is displayed.
  3. Select the UDF module you want to delete.
  4. Click Delete from the Action drop-down menu. A confirmation dialog box opens up.
  5. Click Confirm and the module is deleted. The Delete action is disabled if you do not have permission to delete the file.
