Linking dataflows

The dataflows can be modularized with the help of Link in, Link out and Link out Optimized Nodes. You can break your dataflow in smaller modules, test one module, connect a Link out Node to the end of this module, and use it as input to the next module in the same workbook. This allows you to focus on smaller modules and build a big dataflow gradually.

Using the Link out Node

Follow these steps to use a Link out Node:

  1. Create a dataflow.
  2. Click on the Out category of operations in the Advanced Mode toolbar to display the Link out Node.
  3. Drag the Link out Node to the dataflow canvas.
  4. Connect the last Node of the dataflow to the Link out Node.
  5. Right click on the Link out Node and select Configure menu from the drop-down. The Link Out panel is displayed.
  6. Enter a name in the Link Out Reference Name field for the Link out Node.
  7. If the Only allow linking after execution checkbox is checked, the Link in Node that refers to this Link out Node, uses the result set of the Link out Node. If it is unchecked, the Link in Node executes the dataflow to generate the result set. The Only allow linking after execution checkbox is checked by default.
  8. Click the Save button. Now you have a Link out Node with a name that can be used as an input to a Link in Node in another dataflow.

Using the Link out Optimized Node

The Link out Optimized Node is used when you want to optimize your dataflow, for example in a production environment. Follow these steps to use a Link out Optimized Node:

  1. Create a dataflow and test it.
  2. Click on the Out category of operations in the Advanced Mode toolbar to display the Link out Optimized Node.
  3. Drag the Link out Optimized Node to the dataflow canvas.
  4. Connect the last Node of the dataflow to the Link out Optimized Node.
  5. Right click on the Link out Optimized Node and select Configure menu from the drop-down. The Link Out panel is displayed.
  6. Enter a name in the Link Out Reference Name field for the Link out Optimized Node.
  7. In the Column to export section, select the columns you need in your result set. Optionally click on Select All checkbox to select all the columns.
  8. If the Only allow linking after execution checkbox is checked, the Link in Node that refers to this Link out Optimized Node, uses the result set of the Link out Optimized Node. If it is unchecked, the Link in Node executes the dataflow to generate the result set. The Only allow linking after execution checkbox is checked by default.
  9. Click the Save button. Now you have a Link out Optimized Node with a name that can be used as an input to a Link in Node in another dataflow.
  10. Right click on the configured Link out Optimized Node and select View Optimized Dataflow to see the optimized execution plan of the dataflow.

Using the Link in Node

Follow these steps to use a Link in Node:

  1. Click on the In category of operations in the Advanced Mode toolbar to display the Link in Node.
  2. Drag the Link in Node to the dataflow canvas.
  3. Right click on the Link in Node and select Configure menu from the drop-down. The Link In panel is displayed.
  4. Click on the Dataflow Name field to drop down a list of dataflows in the workbook. Select a dataflow that contains the Link out Node that you want to use as input to this Link in Node.
  5. Click on the Link Out Reference Name field to drop down the list of Link out Node names available in the selected dataflow. Select the one that you want to use as input to the Link in Node.
  6. The Column Schema of the result set from the Link out Node is displayed.
  7. Click on the X before the Column Name to remove it from the result set.
  8. Click on Add Column label to add an empty column name and column type field to the Column Schema.
  9. Click on the empty Column Name field to drop down the list of available column names. Select a column name you want to add to the result set. The corresponding Column Type field is automatically filled in.
  10. If you add a Column Name that already exists in the Column Schema, the existing column name is wiped out.
  11. Once you are satisfied with the Column Schema for the result set, click the Save button.
  12. If a Column Name is left empty, an error message is displayed. You can either fill it up or remove the empty column and click the Save button again.
  13. If there are no errors, the Link in Node is populated with the Link out result set.

Finding the Link out Node connected to a Link in Node

Sometimes you may want to find out the Link out or Link out Optimized Node connected to a Link in Node to investigate that dataflow. Right click on a Link in Node and select the Find the linked node menu item from the drop-down. It opens up the dataflow with the connected Link out or Link out Optimized Node highlighted.
