Publishing a result set to a Jupyter Notebook

You can publish a result set to a Jupyter Notebook for data analysis or visualization of the published rows.

NOTE: To learn more about using Jupyter Notebook see, Using Xcalar Design with Jupyter Notebook.

To publish a result set to a Jupyter Notebook in Dataflow Mode:

  1. Verify that you are in Dataflow Mode. If not, from the top menu bar of Xcalar Design, click the mode switch to display the Dataflow Mode workspace.

  2. There are 2 ways to publish to a Jupyter Notebook:
    1. Starting from a result set:
      1. Click on the Result Set view menu and select Convert to Jupyter DF.
      2. A Jupyter Node is added and connected to the result set Node in the dataflow editor.
    2. Starting from Dataflow Mode toolbar:
      1. Click on the Out category of operations in the Dataflow Mode toolbar to display the Jupyter Node.
      2. Drag the Jupyter Node to the dataflow canvas.
      3. Connect the result set, you want to publish, to the Jupyter Node.
      4. Right click the Jupyter Node and select Configure.
  3. The Jupyter panel is displayed.
  4. Enter the Number of rows you want to publish. The default and the maximum value is 1000.
  5. Rename any column names by clicking on the column name field in Rename columns section.
  6. Click Save to save the configuration. The result set is generated when the Node executes.
  7. Right click on the Jupyter Node and select View Result from the drop-down to view the notebook.
  8. A Rename modal window is displayed for you to rename the Jupyter notebook name. Enter a name and click the Rename button.
  9. The result set is saved to the Jupyter notebook name you chose.
  10. Depending on whether you have an open notebook, the JUPYTER NOTEBOOK window shows one of the following:
    • The Jupyter Notebook dashboard with a file tree, which lists the notebooks created for all workbooks. You can manage your notebooks by opening a notebook for editing or uploading a notebook.

    • The notebook editor with the contents of the open notebook. You can edit the contents as you normally would in Jupyter Notebook.

      IMPORTANT: Do not edit the Xcalar connection code snippet, which is necessary for Xcalar Design to work with the notebook. The code is in the first cell of the notebook. If you accidentally change the code, you can restore the code by following the instructions in Connecting a Jupyter Notebook to an active workbook.
