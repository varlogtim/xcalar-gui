Using parameters in a dataflow

Parameters provide you a convenient way to pass different string values in each execution of a dataflow. You can reuse the same parameter at multiple places in any dataflow in a workbook. Parameters are always of type String and they are local to a workbook.

Typical steps for parametrizing an operation

To parametrize an operation:

  1. Create a parameter.
  2. Assign a value to the parameter.
  3. Pass the parameter to the operation.
Recommendation: If you do not want to create a parameter, you can enter the desired value where a parameter should be placed but the best practice is to define a parameter instead of entering the value directly, where possible.

Parameter naming guidelines

When creating a parameter, remember to select a name that describes the meaning of the parameter.

The following list describes the naming conventions:
  • The name is case sensitive.
  • Length must not exceed 255 characters.
  • Characters must be A-Z, a-z, or 0-9.
  • Cannot contain special characters, such as:
    • hyphen (-)
    • left (() or right ()) parenthesis
    • space
    • underscore (_)

Verify that you are in Dataflow Mode. If not, from the top menu bar of Xcalar Design, click the mode switch to display the Dataflow Mode workspace.

Creating a parameter

To create a parameter:

  1. Click the PARAMETERS button in the top right corner of the Xcalar Design to display the parameters window.
  2. Enter the parameter name in the Input New Parameter field and click the CREATE NEW PARAMETER button.
  3. Click in the Value column next to the new parameter name, and enter the parameter's value. If you don't enter a value for the parameter, it is assigned an empty string.
  4. Values are automatically saved. Click outside the parameters window to close it.

Displaying parameters

When customizing an operation which supports parameters, you can see the list of parameters already defined in the workbook. To display a list of existing parameters and their values, click the PARAMETERS button in the top right corner of the Xcalar Design.

Deleting a parameter

To delete a parameter from the parameters window:

  1. Click the PARAMETERS button in the top right corner of the Xcalar Design to display the parameters window.
  2. Locate the parameter to be deleted and then click the x icon in the Action column.
