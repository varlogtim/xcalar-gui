Using an extension in a dataflow

Xcalar has a framework through which new extensions can be written to augment the features of Xcalar Design. This framework is still in development but Xcalar provides a couple of useful extensions that you can use in your dataflows. Follow these steps to use an extension:

  1. There are two ways of using an extension in the Dataflow Mode - starting from a result set and starting from an Extension Node.
    • Starting from a result set: Display the result set on which you want to apply an Extension. Right click the header of a column and select Extensions to display the Extension configuration panel.

    • Starting from an Extension Node: Click on the Extensions category of operations in the Dataflow Mode toolbar. Drag the Extension Node to the dataflow canvas. Locate the Node with the result set on which you want to apply the Extension, and connect its output to the Extension Node. Right click the Extension Node and select Configure. The Extension configuration panel is displayed.

      NOTE: See Creating and editing a dataflow for more information on connecting Nodes and creating dataflows.

  2. The Extension panel shows a drop-down with installed extension names.
  3. Select an extension name and enter appropriate information in the corresponding fields. The information required depends on the selected extension.

  4. Click Save.

For more information on the Business Rule Inventory extension, see Understanding Business Rules Inventory Extension.

For more information on the Windowing extension, see Understanding Windowing Extension.
