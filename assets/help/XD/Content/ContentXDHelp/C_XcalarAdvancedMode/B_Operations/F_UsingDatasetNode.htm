Using a Dataset Node

A Dataset Node is one of the starting Nodes of a dataflow. It reads the data from a data source and displays it in a result set. For more information on dataset, see Managing datasets.

Follow these steps to use a Dataset Node in a dataflow:

  1. Click on the In category of operations in the Advanced Mode toolbar to display the Dataset Node.
  2. Drag the Dataset Node to the dataflow canvas.
  3. Right click the Dataset Node and choose Configure from the drop-down menu. The Dataset panel is displayed.
  4. Select a dataset to use in the dataflow. If a dataset does not appear in the list or is inactive, close the Dataset panel and load or activate the dataset first.
  5. The Prefix field is already filled in based on the dataset name you choose. You can enter text in this field to choose a different result set name.
  6. Click the Next button. The Column Schema of the dataset is displayed.
  7. Click on X before a column name if you don't want that column in the resultant result set. Or, click on CLEAR ALL field to remove all the column names from the result set.
  8. Click on Add Column to add an empty column name to the Column Schema.
  9. Click on the empty Column Name field to select a column name from the drop-down. The Column Type field is automatically filled up when you select a column name.
  10. Repeat the last two steps to add as many columns as you want.
  11. Once you have added all the columns you need, click on the Save button. This saves the configuration of the Dataset Node and generates the result set when executed.

Changing the source of a dataset Node

Follow these steps to change the source of a dataset:

  1. Right click on the Dataset Node in a dataflow and select Configure menu item from the drop-down.
  2. Select a different dataset as source and click the Next button. The Column Schema of the dataset is displayed.
  3. Click on X before a column name if you don't want that column in the resultant result set. Or, click on CLEAR ALL field to remove all the column names from the result set.
  4. Click on Add Column to add an empty column name to the Column Schema.
  5. Click on the empty Column Name field to select a column name from the drop-down. The Column Type field is automatically filled up when you select a column name.
  6. Repeat the last two steps to add as many columns as you want.
  7. Click the Save button:
    1. If the Dataset Node is not connected to any other Node, under certain conditions, another Dataset panel is displayed for you to map the previous dataset columns to the new dataset columns. Click Save and the configuration is saved in the Dataset Node and generates a result set on execution.
    2. If the Dataset Node's output is connected to another Node, another Dataset panel is displayed for you to map the previous dataset columns to the new dataset columns. This mapping is needed so that the connecting Node understands the new input. The connecting Node's status may turn red if you don't do this mapping. Follow these steps to do the mapping:
      • Click on each of the Previous Columns field and select a column from the drop-down to map to the New Name. Leave it blank if there is no column in the Previous Columns to map to the New Name.
      • Click on the Save button. The configuration is saved in the Dataset Node and generates a result set on execution.
