Using a Table Node

A Table Node is one of the starting Nodes of a dataflow. It reads the data from an already published table and displays it in a result set.

Follow these steps to use a Table Node in a dataflow:

  1. Click on the In category of operations in the Advanced Mode toolbar to display the Table Node.
  2. Drag the Table Node to the dataflow canvas.
  3. Right click the Table Node and choose Configure from the drop-down menu. The IMD Table panel is displayed.
  4. Select a table from the Published Table drop-down.
  5. Select table version section lets you select the version of the table you want to use. The first version of a table always has version 0. The version of the table is incremented by 1 each time there is an update (Insert, Modify, Delete) on the table. Follow these steps to select a version:
    1. Use latest version checkbox is automatically checked for you to use the latest version of the selected table. Alternatively, you can uncheck this checkbox and use -1 in the drop-down below it. Value -1 always points to the latest version.
    2. If you want to use an older version of the table, you can uncheck the Use latest version checkbox and select a version in the drop-down below it.

  6. (Optional) Enter a Filter function in the Filter String to filter the table data. For example, to fetch only those rows that have carrier as UA from the airlines table, enter:

    eq(airlines::Carrier, "UA")

  7. Click the Next button.
  8. The Column Schema of the selected table is displayed. Click on the X before a column name to remove it from the resultant result set.
  9. Click on the Add Column field to add a column to the result set. This adds an empty Column Name and an Column Type field. Click on the empty column name field and choose a column from the drop-down. Column Type is automatically selected based on the column name you choose.
  10. Click the Save button. The Table Node configuration is saved and a new result set is displayed when the Node executes.
  11. If you added a column but did not select the column name, an error message is displayed. You can either select a column name for that column or remove the column and click Save again.
