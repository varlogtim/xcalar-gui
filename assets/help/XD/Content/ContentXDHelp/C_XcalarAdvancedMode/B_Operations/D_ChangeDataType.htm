Changing the data type of a column

In a result set, you can change the data type of a column. Changing the data type of a column creates a new column. For information about the data types supported by Xcalar and the results of casting strings to numeric values, see Data types supported.

IMPORTANT: Changing the data type to and from the float type might cause a value to lose precision. Read Data types supported to understand the effects of type casting a column.

Methods for changing the data type in a result set

You can change the data type of a column in the following ways:

  • Use the Change data type option in the column drop-down list or in the Menu drop-down list when you view the result set.
  • Add a Cast operation to the dataflow.
  • Use a Map function to type cast a column.

Using the Change data type option

To change the data type for one or more columns from the result set, follow these steps:

  1. Display the result set in which you want to set the data type of a column.

  2. Follow one of these steps:

    • Right click the column for which you want to change the data type to display a drop-down menu.
    • Right click Menu in the result set view to display a drop-down list.
  3. Click the Change data type option. A Cast Node is created in the dataflow, and the Cast panel is displayed. For information about configuring the Cast Node, see Configuring a Cast Node.
  4. Execute the Cast Node.

Adding a Cast operation to the dataflow

To add a Cast operation Node to the dataflow, follow these steps:

  1. From the Columns operation category, select the Cast Node.
  2. Add the Cast Node to the dataflow.
  3. Link the Cast Node to the appropriate parent Node.
  4. Right click the Cast Node and click the Configure option. The Cast panel is displayed. For information about configuring the Cast Node, see Configuring a Cast Node.
  5. Execute the Cast Node.

Configuring a Cast Node

To configure a Cast Node, follow these steps in the Cast panel:

  1. Click a column displayed in the Cast panel. You can also type a part of the column name to search for the column. If there is a match, the column name is highlighted.
  2. The Current Name field shows the name of the column selected. You can change it by clicking the down arrow and selecting another column in the drop-down list.
  3. In the New Name field, accept the suggested column name or enter a new name for the column after the data type is assigned to it.
  4. In the New Type, click the down arrow to display a list of data types. Click the one appropriate for the new column.
  5. Repeat steps 2 to 4 for all columns that you want to cast.
  6. Click Save.
  7. If the Cast Node is not executed automatically, execute the Node. Now the result set contains the new columns with the specified data type.

Changing an unknown data type to a valid data type

If the source Node of the dataflow is a dataset that contains an array or object column, you can create columns by pulling fields in the array or object in the Data Browser. For information about pulling fields from the Data Browser to a result set, see Performing tasks in the Data Browser.

To cast the newly created column to a data type, follow these steps:

  1. Right click the Dataset Node, which is the source Node of the dataflow.
  2. Click Configure in the menu. The Dataset panel is displayed, with the current dataset selected.
  3. Click Next in the Dataset panel.
  4. Click Add Column. Enter the new column's name and select the desired data type.
  5. Click Save.
  6. If the Dataset Node is not executed automatically, execute the Node. Now the result set contains the new column with the specified data type.

Using the Map operation

To use the Map operation to cast a column, follow these steps:

  1. From the Columns operation category, select the Map Node.
  2. Add the Map Node to the dataflow.
  3. Link the Map Node to the appropriate parent Node.
  4. Right click the Map Node and click the Configure option. The Map panel is displayed.
  5. Select the type-casting category, and then the function for setting the type.
  6. In the input field, click the down arrow to display a list of columns, then select the column to cast.
  7. Enter the name of the resultant column name.
  8. Click Save.
  9. If the Map Node is not executed automatically, execute the Node. Now the result set contains the new column with the specified data type.
