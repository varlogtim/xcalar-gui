Creating, editing, and deleting a dataflow

This topic describes how to create and edit a dataflow. It also describes how to delete a dataflow that you no longer need. It assumes that you are familiar with the purpose of a dataflow and the components of a dataflow, as described in Understanding dataflows.

Before creating a dataflow

NOTE: You can start creating a dataflow with any Node, but the dataflow cannot be executed successfully if it does not start with a source. This topic assumes that you want to create a dataflow that you can execute.

Each dataflow must obtain its data from a source that resides on the cluster. Therefore, verify that you have at least one source before creating a dataflow. A source can be one of the following items:

  • a dataset created by you
  • a shared dataset created by another user
  • a table created by you or another user

You can also start a dataflow from the result set of another dataflow. For information about linking dataflows, see Linking dataflows.

Creating a dataflow

To create a dataflow, follow these steps:

  1. To create a dataflow, click the plus icon in the dataflow editor. You can also click Create new dataflow if it is displayed, as shown in the following screenshot.

  2. Select the In Node category. The available Nodes that you can use to bring data in are displayed, as shown in the following screenshot:

  3. Click and drag the Dataset or Table Node to the canvas.

  4. (Optional) If you want to replace the default dataflow name, enter the name in the tab title and press Enter,

    The following list describes the naming conventions:
    • The name is case sensitive.
    • Length must not exceed 255 characters.
    • Characters must be A-Z, a-z, 0-9, hyphen (-), underscore (_), or left or right parenthesis.

You are ready to configure the Node, as described in Performing tasks on a Node, or to add more Nodes to the dataflow, as described in Editing a dataflow.

Editing a dataflow

To edit a dataflow, follow these steps:

  1. To add a Node to a dataflow, follow these steps:

    1. Click the Node category name to display the Nodes available in the category.

      Alternatively, enter the name of the Node in the search field. When a drop-down list is displayed in the search field, click the desired Node name. The selected Node is highlighted.

    2. Drag the selected Node to the canvas.

  2. (Optional) If you do not want to keep the default Node name such as Node 1, double-click the Node name and enter the new name. There are no restrictions on the characters for the name. You can also press Enter to start a new line if you want a multi-line Node name displayed under the Node.

    After you finish entering the name, click anywhere outside the Node name field. You can change the Node name again at any time.

  3. To connect two Nodes, click the anchor of one Node and drag the cursor to any part of the other Node. You can start from the output anchor or input anchor. Depending on the anchor you start from, Xcalar Design creates a link to the appropriate anchor of the other Node.

    EXAMPLE:  In a dataflow with a Sort Node and Filter Node, drag the cursor from the output anchor of the Sort Node to anywhere on the Filter Node. The dataflow now has a link from the input anchor of the Filter Node to the output anchor of the Sort Node.

    The Node with the output anchor connected is the parent Node; the Node with the input anchor connected is the child Node.

  4. To change a link, follow one of these steps:

    • To remove a link, click a link. After it is highlighted, click Remove Connection in the pop-up.
    • To change the connection, you can either remove the link and make a new connection, or click the input anchor of the child Node and drag the cursor to a new parent Node.

Multi-selecting Nodes

You can select more than one Node to perform a task such as removing or moving Nodes.

To select multiple Nodes, follow these steps:

  1. Hold down the Shift key when you click each Node.
  2. Drag the cursor to draw a selection box to include the Nodes to be selected.

Removing a Node

To remove a Node, right click the Node to display a menu. Click Remove in the menu. The Node and all the links to and from the Node are removed.

Repositioning or aligning Nodes

For readability, you might want to reposition the Nodes in a dataflow, for example, to avoid having two links cross each other. Repositioning the Nodes does not affect the execution results, which only depend on the configurations and the order of the Node execution.

To move or align Nodes in a dataflow, follow these steps:

  1. To move a single Node, click and drag the selected Node to the desired position.
  2. If you want all Nodes to be properly aligned, right click any Node in the dataflow to display a menu. Click Auto Align in the menu.

Deleting a dataflow

To delete a dataflow, follow these steps:

  1. Click the Dataflow icon to display the Dataflows panel.

  2. In the Dataflows panel, locate the dataflow that you want to delete.

    IMPORTANT: Dataflow deletion is permanent and cannot be undone. You can delete any dataflow, even if it was created and placed in the Published folder by another user. Confirm with other users that they no longer need the dataset before you delete it.
  3. Click the trash can icon to delete the dataflow
  4. When the confirmation prompt is displayed, click Confirm.
