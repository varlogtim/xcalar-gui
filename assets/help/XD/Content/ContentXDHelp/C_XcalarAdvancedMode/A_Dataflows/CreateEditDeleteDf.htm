Creating and editing a dataflow

This topic describes how to create and edit a dataflow. It assumes that you are familiar with the purpose and the components of a dataflow, as described in Understanding a dataflow.

Before creating a dataflow

You can start creating a dataflow with any operation Node, but the dataflow cannot be executed successfully if it does not start with a data source. Therefore, verify that you have at least one data source before creating a dataflow. A data source can be one of the following items:

  • a dataset created by you
  • a shared dataset created and shared by another user
  • a table created by you
  • a table created by another user

For more information about importing data into datasets and tables, see Importing data into Xcalar Design.

Opening the dataflow editor

To open the dataflow editor, verify that you are in the Dataflow Mode. If not, from the top menu bar of Xcalar Design, click the mode switch to display the Dataflow Mode workspace. Then, from the sidebar, click the Dataflows icon, as shown in the following screenshot.

Creating a dataflow

To create a dataflow:

  1. Click the plus icon in the dataflow editor or click the Create new dataflow button if it is displayed, as shown in the following screenshot.

  2. Select the In Node category. This displays all the Nodes that you can use to bring data in your dataflow:

  3. Click and drag the Dataset or the Table Node to the dataflow canvas.

  4. (Optional) If you want to replace the default dataflow name, enter the name in the tab title and press Enter,

    The following list describes the naming conventions:
    • The name is case sensitive.
    • Length must not exceed 255 characters.
    • Characters must be A-Z, a-z, 0-9, hyphen (-), or underscore (_).

You now have a dataflow with one operation Node and it's listed in the Home folder of the Dataflows panel. At this point, either you can configure the Node in your dataflow, as described in Configuring an operation Node, or continue adding more operation Nodes, as described in Editing a dataflow.

Editing a dataflow

To edit a dataflow:

  1. Find the dataflow, you wish to edit, in the Dataflows panel and click it.
    The dataflow is displayed in the dataflow canvas.

  2. To add a Node to a dataflow:

    1. Click a Node category name in Dataflow Mode toolbar to display the Nodes available in that category.

      Alternatively, enter the name of the Node in the search field in the Dataflow Mode toolbar. When a drop-down list is displayed in the search field, click the desired Node name. The selected Node is highlighted.

    2. Drag the selected Node to the dataflow canvas.

  3. (Optional) If you do not want to keep the default Node name such as Node 1, double-click the Node name and enter a new name. There are no restrictions on the characters for the name. You can also press Enter to start a new line if you want a multi-line Node name.

    After you finish entering the name, click anywhere outside the Node name field. You can change the Node name again at any time.

  4. To connect the two Nodes, click the anchor of one Node and drag the cursor to any part of the other Node. You can start from the output anchor or the input anchor. Depending on the anchor you start from, Xcalar Design creates a link to the appropriate anchor of the other Node.

    EXAMPLE:  In a dataflow with a dataset Node and a Filter Node, drag the cursor from the output anchor of the dataset Node to anywhere on the Filter Node. The dataflow now has a link from the output anchor of the dataset Node to the input anchor of the Filter Node.

    In a link, the Node with the output anchor connected is the parent Node, and the Node with the input anchor connected is the child Node.

  5. To change a link:

    • To remove a link, click a link. After it is highlighted, click Remove Connection in the pop-up.
    • To change the connection, you can either remove the link and make a new connection, or click the input anchor of the child Node and drag the cursor to a new parent Node.

Multi-selecting Nodes

You can select more than one Node to perform a task such as removing or moving.

To select multiple Nodes, do one of the following:

  • Hold down the Shift key when you click each Node.
  • Drag the cursor to draw a selection box to include the Nodes to be selected.
  • To select all Nodes, right click anywhere on the dataflow canvas and select Select All.

Removing a Node

To remove a Node, right click the Node and select Remove. The Node and all the links to and from the Node are removed. If you want to remove all Nodes from a dataflow, deselect all Nodes, right click on the dataflow canvas and select Remove All.

Repositioning or aligning Nodes

For readability, you might want to reposition the Nodes in a dataflow, for example, to avoid having two links cross each other. Repositioning the Nodes does not affect the execution result, which only depends on the configuration and the order of the Node execution.

To move or align Nodes in a dataflow:

  • To move a single Node, click and drag the selected Node to the desired position.
  • If you want all Nodes to be properly aligned, click anywhere on the dataflow canvas and select Auto Align.
NOTE:  Xcalar Design saves all your work automatically. You do not need to manually save your changes.

Adding Comment to a dataflow

If you want to add a comment in your dataflow, deselect all Nodes, right click on the dataflow canvas at the point where you want to add the comment and select New Comment. A text box is created for you to add your comment. You can drag the bottom right corner to change the size of the text box.

To copy, cut or remove the text box, right click anywhere in the text box and select the desired operation.
