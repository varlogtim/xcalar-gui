Configuring settings for dataflows

This topic describes the purpose of the dataflow settings and how to configure the settings.

Understanding dataflow settings

The dataflow settings affect how dataflows behave in all your workbooks. The following list describes the effects of enabling the settings on dataflows:

  • Auto execution: This setting causes a Node to be automatically executed after you finish configuring it. Without auto execution, after you configure a Node, you must manually execute the Node if you want to see the result.
  • Auto preview: This setting displays the result set immediately after you execute an operation. If you execute all operations, the result set of the last operation is displayed.
  • Show progress details: This setting displays the progress of the execution and display the statistics above the operation Node after the execution. If this setting is disabled during execution, you can still display the statistics by enabling this setting after the execution.
  • Show configuration information: This setting displays the configuration information below each Node. For example, the configuration information for a Dataset Node indicates which dataset provides the source data for the dataflow.

Configuring the dataflow settings

To configure the dataflow settings:

  1. Click the dataflow settings button, as shown in the following screenshot. The Dataflow Settings modal window is displayed.

  2. Click the appropriate check box in the Dataflow Settings modal window, then click Save.
