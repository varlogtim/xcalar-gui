Xcalar SQL keyboard shortcuts

The following table provides a list of Xcalar SQL keyboard shortcuts.

To do this action

Windows

Mac

Runs the selected query, or if nothing is selected, the entire query.

Ctrl+e, Alt+x

command+e, alt+x

Cancels the query's execution.

Ctrl+Alt+c

F6

Changes the selected text to lowercase.

Ctrl+Shift+l

command+shift+l

Changes the selected text to uppercase.

Ctrl+Shift+u

command+shift+u

Toggle between commenting and uncommenting the text.

NOTE: When uncommenting text, only one instance of double-hyphens is removed at a time.

Ctrl+-

control+-

Moves the cursor one word to the left.

Ctrl+left arrow

command+left arrow

Moves the cursor one word to the right.

Ctrl+right arrow

command+right arrow

Extends the selection one word to the left.

Ctrl+Shift+left arrow

command+shift+left arrow

Extends the selection one word to the right.

Ctrl+Shift+right arrow

command+shift+right arrow

Scrolls the text up one line.

Ctrl+up arrow

command+up arrow

Scrolls the text down one line.

Ctrl+down arrow

command+down arrow

Inserts a blank line above the cursor.

Ctrl+Enter

command+return

Inserts a blank line below the cursor.

Ctrl+Shift+Enter

command+shift+return

Deletes a word to the left of the cursor.

Ctrl+Backspace

command+delete

Deletes a word to the right of the cursor.

Ctrl+Delete

command+ shift+delete

Opens the Search window.

Ctrl+f

command+f

Opens the Replace window.

  1. In the Replace field, enter the existing text and press Enter.
  2. In the With field, enter the replacement text and press Enter.
  3. From the Replace? options, click either Yes, No, or Stop.

Ctrl+h

command+option+f

Finds the next text match from the Search field.

Ctrl+Alt+g

F3
Finds the previous text match from the Search field.

Ctrl+Shift+Alt+g

shift+F3

Deletes a line.

Ctrl+Shift+K

command+shift+K
