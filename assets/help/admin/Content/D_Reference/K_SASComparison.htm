SAS-equivalent functions

This section lists the Xcalar functions that accomplish the same tasks as the SAS functions. If an equivalent Xcalar function is not available, this section suggests an alternate method. Also, if the Xcalar function works differently than the corresponding SAS function, a brief description of the Xcalar function is provided.

SAS function category: Operators

SAS function SAS description Xcalar Map function Xcalar syntax
+ Plus (Addition) add map(add(operand1,operand2))
- Minus (Subtraction) sub map(sub(operand1,operand2))
* Multiply mult map(mult(operand1,operand2))
/ Divide div map(div(operand1,operand2))
** Exponentiation pow map(pow(operand1,operand2))
= Equal eq map(pow(operand1, operand2))
=> Greater than or equal to ge map(ge(operand1, operand2))
> Greater than gt map(gt(operand1, operand2))
=< Less than or equal to le map(le(operand1, operand2))
< Less than lt map(lt(operand1, operand2))
^= or NE Not equal to neq map(neq(operand1, operand2))
|| Concatenates concat map(concat(operand1,operand2))

SAS function SAS description Xcalar Map function Xcalar syntax

SQRT Returns a positive square root sqrt map(sqrt(operand))

SAS function category: Logical

SAS function SAS description Xcalar Map function Xcalar syntax
AND Returns TRUE if all of its arguments are TRUE and map(and(operand1,operand2))
IF Specifies a logical test to perform if map(if(condition, return value, return value if false))
NOT Reverses the logic of its argument not map(not(operand))
OR Returns TRUE if any argument is TRUE or map(or(operand1,operand2))

SAS function category: Character

SAS function SAS description Xcalar Map function Xcalar syntax
COMPBL Removes multiple blanks between words in a character string replace map(replace(string, " ", ""))
COMPRESS Removes specific characters from a character string replace map(replace(string, char, ""))
INDEX Searches the source for the character string specified by the excerpt find

map(find(string to search, string to find, starting index, ending index))

The search is case-sensitive.
LENGTH Returns the number of characters in a text string len map(len(operand))
SUBSTR Returns the specified number of characters starting at the specified position. substring map(substring(string, start_index, end_index))
TRIM Removes trailing blanks from character expression and returns one blank if the expression is missing strip

map(strip(operand))

Strip leading and trailing white space from a string.
