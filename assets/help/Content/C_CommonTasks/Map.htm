Using the map function to create new values from selected columns

The map function is divided into categories of functions that you use to create new values from one or multiple selected columns. These new values enable you to derive meanings from your dataset. For example, you can perform a trigonometric calculation from values in two selected columns to create a third column that contains the results of the calculation, or you can generate a column of unique values from a selected column.

This section describes the steps required to perform a map function. For descriptions of all map functions, see Functions categorized under the map function.

Creating a column by the map function

Follow these steps to create a column by the map function:

  1. Right click the header of the column that you want to create new values from. In the drop-down menu, select Map…. A modal window is displayed.

  2. In the Map… modal window, follow these steps:

    1. Select the category from the Category drop-down menu.
    2. Select the function from the Function drop-down menu.
    3. In the Argument List, enter the name of the column for which you want to perform the map function.
    4. In the Argument List, enter a column name or a value as an argument. Some functions require one argument while others require multiple arguments.

    5. Accept the suggested name for the new column or type the desired name.

    Expand and read the following list that describes the rules and tips for entering information in a modal window launched by a Xcalar Insight function.
    • By default, the name of the column from which you start the function is displayed as an argument. If you want to use another column as an argument, you can change the argument to the name of another column.
    • When typing a column name as an argument, always start with the dollar sign ($).
    • If you provide a name for the column created by the function, simply type the column name without the dollar sign ($).
    • To avoid mistakes when typing a column name as an argument, when your insertion point is in the argument field, move the cursor to the active table and click anywhere in the desired column. The column's name is automatically entered in the argument field.
    • You can undo any typing with the undo key or key combination that you normally use in your browser.
    • If the argument list includes check boxes in the Empty column, leave the check boxes unselected unless the argument is an empty string. Selecting the check box indicates that you indeed want to pass an empty string as an argument to the function.
  3. Click Apply.
