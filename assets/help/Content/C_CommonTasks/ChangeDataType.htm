Changing the data type for a column

By default, the data type for each column is string. You can change the type to Boolean, Integer, or Float.

Supported integer types and floating point values

The following integer types are supported:

  • Signed and unsigned 32-bit integers
  • Signed and unsigned 64-bit integers

Both 32-bit and 64-bit floating-point values are supported.

Different ways to change the data type

Right-click the column heading to display a drop-down menu. Then you can use one of the following methods to change the data type:

  • Select the Change data type option in the column drop-down menu. This method does not create a column.
  • Select the Map option in the column drop-down menu. This method keeps the existing column intact and creates a column with the new data type.

Using the Map function to change the data type

After you select the Map option in the column drop-down menu, the Map modal window is diaplayed. Follow these steps to change the data type:

  1. In the Category field, select Type-Casting Functions.
  2. In the Functions field, select the desired data type.

  3. In the Argument list, you can accept the default name or enter a name for the new column that is created with the new data type.

    If the desired data type is integer, you have the option of enter the base. For example, for binary, enter 2 as the base.

  4. Click Apply. A new column is created with the new data type.
