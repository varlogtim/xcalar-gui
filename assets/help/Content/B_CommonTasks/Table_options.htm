Table options

You can right click anywhere in the table title bar or click the triangle in the lower-right corner of the table title bar to display a drop-down menu. The menu presents table options for manipulating the entire table. This section explains the purpose of each table option.

Archive Table

You can archive a table that you do not use often. The archived table does not show up in the worksheet, but can be added back at any time from the Tables list. Also, the query graph of an archived table is removed from the query graph viewer.

Hide Table

You can hide a table so that the columns are not viewable. A small portion of the table is still shown in the worksheet, enabling you to click the table title bar to display the table options. One of the options in the menu from a hidden table is Unhide Table, which returns the table to view again.

Drop Table

You can drop a table you no longer need.

NOTE: You cannot undo a table deletion.

Export Table

You can export a table or selected columns of a table to a predefined target.

Copy Column Names

You can copy all column names to the clipboard. Suppose you have columns named MonthDayYear, DayOfWeek_integer, and DepTime_integer. The following array of names can be pasted from the clipboard:

["MonthDayYear","DayOfWeek_integer","DepTime_integer"]

Deleting All Duplicates

If you have created columns by duplication, you can delete all the duplicate columns.

Quick Aggregates

The Quick Aggregates option is applicable if the table contains data of Float or Integer type. Clicking Quick Aggregates displays a pop-up menu with the following options:

Aggregate Functions Displays results of common aggregate functions performed on each column. The functions are sum, avg, min, max, and count. If a table column's data type is not Integer or Float, the aggregate function returns N/A for the column.
Correlation Coefficient Displays the correlation coefficient between each pair of columns containing values of data type Integer or Float.

Smart Type Casting

You can change the data type for one or multiple columns. For more information about smart type casting, see Changing the data type for a column.

Sort Columns

You can arrange the table columns by sorting the column headers in ASCII order.

Resize All Columns

You can resize all columns at once. For more information about resizing a column, see Resizing a column.
