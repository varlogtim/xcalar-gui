Joining tables

You can join tables in the same worksheet. The join function works in the same way as the SQL join clause.

NOTE: You can join two tables or join a table to itself (that is, perform a self join). You cannot use one operation to join more than two tables.

Understanding the multi-clause join function

By default, the join function uses one key from each table for joining. That is, you can select one column from each table by which the tables are joined.

You can, however, perform a multi-clause join, which is a join function that uses pairs of join keys.

Steps for joining tables

Follow these steps to join tables:

  1. Right click the header of a column in one of the tables that you want to join. A drop-down menu is displayed.

  2. Select Join… in the menu. The Join Tables modal window appears.

  3. Type the name of the table to be created as a result of the join function.

  4. Select the type of join function to perform. By default, Inner join is selected as the type of join operation. The following types of join operations are supported:

    • Inner join
    • Left outer join
    • Right outer join
    • Full outer join

  5. The tables to be joined are shown as Left Table and Right Table.The columns are displayed under the table name. To join a table instead of the one being displayed, click the table name to display a drop-down list of tables and select another table. To perform a self join, select the same table for Left Table and Right Table.

  6. Decide whether you want to perform a single-clause join or multi-clause join. For information about multi-clause join, go to Performing a multi-clause join.

    For each table displayed, verify that the highlighted column is the key by which you want to perform the join. Select another column if necessary.

    Alternatively, you can click Smart Suggest to determine whether joining by a suggested column would give you more insight into your data. Using Smart Suggest is meaningful only when you have selected one key because the suggestion is based on your selection.

    EXAMPLE: This example is for an inner join. The left table contains two columns: Carrier (airline carrier code) and Description (airline carrier full name). The right table contains three columns named DepTime (departure time), ArrTime (arrival time), and Carrier (airline carrier code). In the Join Tables modal window, if you select the Carrier column in the Left Table, Xcalar Insight's Smart Suggest feature automatically selects the Carrier column in the Right Table because Carrier is the join key that would provide the best result. The newly created table contains all columns from both tables and only rows with a carrier code that exists in both tables.
  7. Click JOIN TABLES.

A new table is created as a result of joining the tables with the keys you selected.

Performing a multi-clause join

Follow these steps to start a multi-clause join:

  1. Follow the same steps as those for a single-clause join. When you are ready to select the join keys, click the icon in the upper right corner of the Join Tables modal window, as shown in the following sample screen.

  2. In the Multi Clause Join section of the Join Table modal window, specify the keys on which the join should be performed. You can drag the column from each table and drop it in the space in the Multi-Clause Join section. The following screen shows that the join function uses two pairs of join keys.

  3. Click JOIN TABLES.

You can perform a join operation for two tables in a similar way as you would by using a SQL join clause.
