Creating a table

This section assumes that you have created one or multiple datasets and you are familiar with pointing to datasets. Also, this section assumes that you are viewing the worksheet where you want to create the table. See Understanding workbooks and worksheets for information about how to switch between worksheets.

NOTE: Tables can be created as a result of table operations. For example, joining two tables results in a new table. This section is about creating a table from adding columns from a dataset.

Follow these steps to create a table from the data in a dataset:

  1. Select the Data Stores tab.
  2. Click a dataset in the left panel.

  3. In the center panel, you can create a table in one of the following ways:

    • Click one or multiple column headings to select the columns that form the table. Clicking a selected column heading deselects the column.
    • Select all the columns to create the table.
    • Create a table without any columns. You can select the columns after the table is created.

  4. (Optional) If you want to create an additional table from another dataset, click another dataset in the left panel. Then repeat step 3. An additional table name is displayed in the right panel (DATA CART panel).

  5. (Optional) In the right panel (DATA CART panel), edit the table name.

    The following list describes the naming conventions for a table:
    • The name can consist of any alphanumeric characters, space, and special characters except the following characters:
      • asterisk (*)
      • hashtag (#)
    • The name must not be a duplicate of an existing table name.
    • It must consist of 1 to 255 characters.

  6. In the right panel (DATA CART panel), click SEND TO ACTIVE WORKSHEET. All tables shown in the data cart are sent to the same worksheet.

For more information about the worksheet window after you create a table, see Worksheet window with tables. You are now ready for transforming your data by using the various functions in Xcalar Insight.
