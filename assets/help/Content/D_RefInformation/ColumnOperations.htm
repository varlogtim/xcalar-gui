Operations for manipulating values in a column

Split column

You can split a column if it contains strings. You cannot split a numeric column.

Enter the delimiter in the Split Column By field. The delimiter is the character that separates the data to be split. Optionally, enter the number of splits. For example, if the date column contains dates in the mm/dd/yy format, you can enter the slash (/) as the delimiter and 1 as the number of splits. Because you specify one split, two columns are created. One has the mm as the heading and the other has the dd/yy as the heading..

Change data type

You can change the data type to Boolean, Integer, Float, or String.

Sort

Values can be sorted alphabetically or numerically. You can change the data type to integer before sorting a column numerically.

Filter

The following conditional functions are supported:

  • And
  • Between
  • Contains
  • Eq
  • Exists
  • Ge
  • Gt
  • If
  • IfInt
  • IfStr
  • IsBoolean
  • IsFloat
  • IsInteger
  • Le
  • Like
  • Lt
  • Not
  • Or
  • Regex

Group By

The following aggregate functions are supported:

  • Avg
  • Count
  • Max
  • MaxInteger
  • Min
  • MinInteger
  • Sum
  • SumInteger

Map

The following categories of functions are supported:

  • Arithmetic
  • Bitwise
  • Conditional
  • Conversion
  • Miscellaneous
  • Trigonometric
  • Type-Casting
  • User-Defined

Join

The Join operation creates a new table based on the columns you select from two tables. It works in the same way as a SQL join.

Profile

Profile presents a bar graph. Each bar shows the number of occurrences of each value found in the column.
