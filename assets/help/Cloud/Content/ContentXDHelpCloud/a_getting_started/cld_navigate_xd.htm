Navigating the online help

This section provides an overview of the Xcalar Design online help documentation.

The Xcalar Design online help provides detailed instructions for accessing your Xcalar cloud account and for performing the tasks associated with developing data models and analyzing data in Xcalar Design.

It also contains conceptual and detailed reference material, including the Xcalar supported SQL ANSI standard features and subfeatures, instructions for working with third-party applications inside and outside of Xcalar, and contains information about the Xcalar Design tutorials, walkthroughs, and other resources for learning Xcalar Design.

Xcalar recommends familiarizing yourself with the contents of this online help by browsing through the content descriptions below, the main sections in the online help sidebar, and using the Search tool.
