Xcalar Design overview

Xcalar Design is a browser based application for designing and executing dataflows and queries to analyze data. The Xcalar Design integrated development environment (IDE) has two workspaces:

  • SQL Mode

    • In this workspace, you develop your queries using SQL query statements and user-defined functions (UDFs) written in Python.

    It contains features and functions that enable you to import your data into tables, apply SQL queries to those tables for analysis, create tables from result sets for more analysis, and export your results to files. The Xcalar Design SQL Mode workspace also provides features for analyzing your query statements and extensions that expand the Xcalar SQL syntax.

  • Dataflow Mode

    • In this workspace, you develop your dataflows using visual components, SQL query statements, and UDFs written in Python.

    It contains features and functions that enable you to import your data into datasets or tables, apply operations to those datasets or tables for analysis, create tables from result sets for more analysis, and export your results to files. This workspace also augments the SQL Mode workspace by providing features and functions for monitoring and troubleshooting your SQL queries.

    This workspace is intended for users who prefer to work with graphical components to perform operations on their data for analysis.

Both the workspaces are independent of each other, and the dataflow or the query developed in one mode is not visible in the other mode. However, there are a few exceptions:

  • You can develop a query in SQL Mode and convert it into a dataflow which can then be viewed and debugged in the Dataflow Mode.
  • You can develop a special dataflow called SQL Function in the Dataflow Mode which can then be invoked by a SQL snippet in the SQL Mode.
