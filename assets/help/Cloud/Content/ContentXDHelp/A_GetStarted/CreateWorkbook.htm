Creating and activating a workbook

This topic describes how to create and activate a workbook in Xcalar Design. Creating a workbook is the first step in designing your data model in Xcalar Design. Workbook can be visualized as a container where you design your data models and execute them.

After you log in to the Xcalar Design, you see the Workbook Browser window. The Workbook Browser is the place where you create and manage your workbooks. It displays a workbook card for every workbook you have.

Creating a workbook

To create a workbook:

  1. Log in to Xcalar Design. The Workbook Browser is automatically displayed. If it is not displayed, click the Workbooks icon in the upper left corner of the window.

  2. In the New Workbook card, click Create New Workbook. A new workbook is created and a corresponding new workbook card is displayed in the Workbook Browser.

  3. To name the new workbook and add a description:

    1. Click the ellipsis icon in the new workbook card and then click Edit Name and Description.

      The following list describes the naming conventions:
      • The name is case-sensitive.
      • Length must not exceed 255 characters.
      • First character must be a letter of the alphabet.
      • Characters must be A-Z, a-z, 0-9, hyphen (-), underscore (_), or space.
      • Must not be one of the following reserved names:
        • udf
        • udfs
        • preupgrade
    2. In the Edit Workbook modal window, enter the workbook name.

    3. Enter a workbook description. This description is displayed on the workbook card in the Workbook Browser, enabling you to make the correct workbook selection.

    4. Click Confirm.
NOTE: The newly created workbook is in the inactive state. You must activate the workbook before you can use it.
TIP: If the complete description is too long to fit on the workbook card, move the cursor over the description to display a tooltip with the complete description.

Understanding the workbook activation

A workbook is inactive right after its creation. In the Workbook Browser, the card for an active workbook is blue, and the card for an inactive workbook is white. Logging in and out of Xcalar Design does not affect the state of a workbook. However, if the Xcalar cluster is stopped and then started again, all workbooks become inactive and you must activate them again before working on your data.

Multiple workbooks can be active at the same time and you can display them in different browser tabs. However, you cannot display the same workbook in two browser tabs.

Activating a workbook

You must activate a workbook before you can use it. For more information about workbook activation, see Understanding the workbook activation.

To activate a workbook:

  1. Log in to Xcalar Design. The Workbook Browser is automatically displayed. If it is not displayed, click the Workbooks icon in the upper left corner of the window.
  2. Click anywhere except the ellipsis icon in the inactive workbook's card to activate and open it.

For information about other tasks related to workbooks, see Managing workbooks.
