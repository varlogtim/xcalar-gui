Selecting a connector

This topic describes the steps required to select a connector for importing data in Xcalar Design.

IMPORTANT: Access and prepare your data source as described in Accessing and preparing a data source before you select a connector.

To select a connector:

  1. Verify that you are in Dataflow Mode. If not, from the top menu bar of Xcalar Design, click the mode switch to display the Dataflow Mode workspace. You can also select a connector from the SQL Mode but then you can only import into a table.

  2. From the sidebar, click the Sources icon and then the Tables icon or the Datasets icon depending on whether you want to create a table or a dataset. You can use a table and a dataset in the same way to begin a dataflow.

    The following screenshot shows the locations of the Tables and Datasets icons:

  3. In the Tables or Datasets panel, click Import Data Source.

    The Source window is displayed with the following connectors:

    File Browser - This connector uses the private Amazon S3 bucket created for you in the Xcalar cluster as the source. To list the content of this S3 bucket, click File Browser. Then, to upload a file from your computer into this S3 bucket, click Choose a file to upload or drag a file to the upload area . To delete a file from the bucket, right click on a file name and select Delete File.

    Amazon S3 Location - This connector enables you to use your own Amazon S3 bucket as the source.

    Database Location - This connector enables you to select a database as the source.

    More Connectors - This option gives you more choices of connectors.

    For more information on connectors, see Understanding the connector types.

  4. Select the connector where your data source is located.
