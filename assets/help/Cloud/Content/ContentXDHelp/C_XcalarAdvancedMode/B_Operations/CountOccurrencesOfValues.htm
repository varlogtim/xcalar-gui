Using Profile to find the distribution of values in a column

Distribution of values in a column provides interesting insights into the data for further analysis. Xcalar Design provides you a simple way to find the distribution of each value in a column, through the Profile option.

IMPORTANT: The Profile option is available in Dataflow Mode only. Verify that you are in Dataflow Mode. If not, from the top menu bar of Xcalar Design, click the mode switch to display the Dataflow Mode workspace.

Displaying the distribution of values in a column

To find the distribution of values in a result set column:

  1. In a result set, right click the header of the column for which you want to find the distribution.

  2. From the drop-down, select Profile. The Profile window containing a bar graph is displayed with the distribution of values in the column.

TIP: It might take a while for the bar graph to display. You can close the window and re-select the Profile column option later to view the finished bar graph.
NOTE: Using Profile to find the distribution of values does not create a result set. If you need to create a result set to show the frequency of each value, use the Count function in the Group By panel. For more information about Group By functions, see Grouping data in a result set.

Sorting the graph in the Profile window

You can sort the graph by choosing the sort options above the graph.

Displaying the frequency with range bucketing

If the data type of a column is Integer or Float, you can use the Range bucketing drop-down to determine how the bars or slices are displayed:

  • To count the occurrences of single values, select Single. Each bar or slice represents the frequency of one value.
  • To count the occurrences of the values that fall in a range, select Range and enter the range size. Each range is represented by a bar or a slice. Enter a numeric value in the Range field and then press Enter. For example, 0.5 and 20 are both valid values.
  • To count the occurrences of the values that fall in a log scale range, select Range (log scale). This automatically divides the whole range of values over a log scale. For example, if your values are ranging from -5 to + 11, it will automatically create the following ranges: -10 to -1, -1 to 0, 0 to 1, 1 to 10, 10 to 100. Each range is represented by a bar or a slice.
  • To instruct Xcalar Design to select the range size so that all bars or slices can fit in the window, select Fit All.
NOTE: A bar or slice is displayed only if there is at least one occurrence of the value represented by the bar. For example, if the range size is 50, and there are no occurrences of any value between 0 and 50, the graph does not display a bar or slice for this range.

Types of charts in the Profile window

You can toggle the Graph switch in the Profile window to display either a bar or a pie chart.

Skip to a specific value

You can skip to a specific bar in a bar chart by entering a bar number in the Skip to field. This is useful when all the bars don't fit in one window.

Specifying the decimal places for field values

If the data type of a column is Integer or Float, you can specify the number of decimal places for each displayed value in the bar or the pie chart. For example, if you specify 0 decimal places, the values are displayed as 1, 2, and so on. If you specify 2 decimal places, the values are displayed as 1.00, 2.00, and so on.

Specifying the number of results to be displayed

If you select Fit All in the Range bucketing drop-down list, you cannot control the number of results to be displayed. Otherwise, you can increase or decrease the number of results displayed by clicking on the up and down arrows in Showing x results. The number is changed by 10.

NOTE: The bar showing the frequency of the null value is displayed in the bar graph or pie chart of each page. It is not counted in the number of results shown. The number of results reflects only the non-null values.

Displaying counts or percentages

By default, the frequency of values is displayed as a count. Click a bar or slice to change the count to a percentage.

Hovering over a bar or slice displays a tooltip, which contains the field value (either as a single value or as a range) and the percentage or the frequency.

EXAMPLE: Suppose the value 1 in the selected column appears 3,000 times, the Frequency: 3,000 is displayed on the bar. Click anywhere in the bar and the number changes to 20%, which means that 20% of the values in the column are 1.

Filtering by using the Profile window

The Profile window provides a convenient way to filter or exclude a value in the result set after viewing the frequencies of various values in a column. You can also filter or exclude a range of values.

NOTE: To filter or exclude a value selectable in a result set column or to filter or exclude a value based on a condition, follow the instructions in Filtering in a result set column.

To filter or exclude a value through the Profile window:

  1. Place the cursor in the bar or slice for the value to be filtered or excluded. Drag in any direction inside the bar or slice. To select multiple bars or slices, repeat this step for each bar or slice.

    TIP: If the bars or slices are contiguous, move the cursor across them.

    After you finish selecting, a drop-down menu displays the options for you to filter or exclude the values:

  2. Select Filter this value in the drop-down to include only this value (or a range of values). Select Exclude to exclude this value (or a range of values).
  3. Once you select an option from the drop-down menu, the Profile window closes and the Filter panel is displayed.

  4. For example if you click Profile on column DayOfWeek in the airlines result set and select Filter this value on value 1, the Filter panel is displayed with Filter Function as eq, Operand 1 as DayOfWeek and Operand 2 as 1. You can change these values and add additional conditions to this filter. Clicking Save on the Filter panel updates the result set with this filter. Downloading the Profile window image

To download the Profile window image:

  1. Hover over the bar graph or pie chart in the Profile window to display the Download icon on the top right corner of the graph.

  2. Click the Download icon to start the download. A file named profile.png is downloaded to your computer.
